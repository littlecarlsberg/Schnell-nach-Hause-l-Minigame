# Schnell nach Hause 🏠

Ein minimalistisches 2D-Jump'n'Run im Berliner Neon-Stil.

![Schnell nach Hause](screenshot.png)

## Spielprinzip

Nach einem langen Tag möchtest du nur noch nach Hause. Laufe durch die nächtlichen Straßen Berlins, sammle Coins und Pizza, springe über Hindernisse und erreiche die Tür zu deiner Wohnung!

## Features

- 🎮 **3 Level** durch Kreuzberg, Tempelhof und Schöneberg
- 🌃 **Berlin-Neon-Ästhetik** mit Low-Poly PS1/N64-Stil
- 🚗 **Autos im Hintergrund** für urbanes Flair
- 🪙 **Coins & Pizza-Power-Ups** zum Sammeln
- 🏆 **Highscore-System** mit lokalem Speicher
- 📱 **Touch-Steuerung** für Mobile
- 🔊 **Retro-Soundeffekte**

## Steuerung

| Aktion | Tastatur | Mobile |
|--------|----------|--------|
| Links | ← / A | ◀ Button |
| Rechts | → / D | ▶ Button |
| Springen | Space / ↑ / W | ▲ Button |

## Spielen

### Lokal
```bash
git clone https://github.com/DEIN-USERNAME/schnell-nach-hause.git
cd schnell-nach-hause
# Öffne index.html im Browser
```

## Tech Stack

- Pure JavaScript (kein Framework)
- HTML5 Canvas
- Web Audio API für Sound
- CSS3 für UI
